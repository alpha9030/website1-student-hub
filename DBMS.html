💡 Recommended Project Ideas

🟢 Beginner Projects
  • ER Diagram for Library: Design a complete ER Diagram representing tables, keys, and relational cardinality for a university library.
  • Basic SQL Schema for Store: Write basic SQL DDL commands to create store tables, setting constraints and primary/foreign keys.
  • SQL Inventory Tracker Schema: Design a relational schema with tables for items, suppliers, and transactions with key constraints.
🟡 Medium Projects
  • E-Commerce Database Schema: Establish tables, indexes, and constraints for users, shopping carts, orders, and products.
  • Normalizer Converter Utility: Write a script in Python/Java to verify functional dependencies and check for 3NF/BCNF violations.
  • Hotel Booking Schema & Functions: Establish tables for rooms, guests, bookings, and payments, including check-in date validation triggers.
🔴 Advanced Projects
  • DBMS Engine from Scratch: Program a minimal file-based DBMS engine supporting basic SELECT, INSERT, and WHERE filters in Python/C++.
  • Banking System Transactions Simulation: Model a database system simulating concurrency issues, deadlocks, and ACID transaction commits.
  • Database Replication Engine: Write a Python script that parses transaction logs from one DBMS database and replicates transactions to a standby server.
