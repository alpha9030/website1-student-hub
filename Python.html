💡 Recommended Project Ideas

🟢 Beginner Projects
  • Basic Web Scraper Script: Write a python script utilizing requests and BeautifulSoup to scrape and save headline articles from blogs.
  • Interactive CLI Weather App: Develop a console utility that queries open weather APIs and displays city statistics.
  • Desktop Notification Script: Build a script that runs in the background and sends system notifications or alarms at specified times.
🟡 Medium Projects
  • To-Do List REST API: Build a functional back-end server utilizing Flask/FastAPI supporting CRUD operations and JSON responses.
  • Custom File Renaming Script: Write a filesystem tool utilizing the os/pathlib module that renames hundreds of files according to dates.
  • PDF Metadata Extractor: Develop a CLI tool using PyPDF2 that extracts author details, page counts, and encrypted states of target PDFs.
🔴 Advanced Projects
  • Real-time Machine Learning Classifier: Program a classification model utilizing Scikit-Learn that evaluates inputs and returns predictions in real-time.
  • Multiplayer Tic-Tac-Toe via WebSockets: Construct a network lobby and socket client allowing real-time board gameplay between two peers.
  • Custom HTTP Banner Grabber: Create a network security script that checks open ports and parses server responses to detect software versions.
