💡 Recommended Project Ideas

🟢 Beginner Projects
  • Sorting Visualizer: Build a simple web application that shows sorting algorithms (bubble, insertion, selection) running visually.
  • Simple Contact Manager: Create a terminal application that uses hash tables or binary trees to store and search contacts.
🟡 Medium Projects
  • Maze Solver: Write a program that generates a maze and uses BFS/DFS or Dijkstra's algorithm to find the shortest path.
  • Text Editor Undo/Redo: Implement a stack-based undo/redo feature for a simple text input tool.
🔴 Advanced Projects
  • File Compressor (Huffman Encoding): Build a utility that compresses and decompresses text files using Huffman trees.
  • Social Network Graph: Design a mini social network system that models connections as graphs, offering degree-of-separation and mutual friend recommendations.
