💡 Recommended Project Ideas

🟢 Beginner Projects
  • School Directory Database: Create a schema representing students, teachers, and classrooms tables, query with basic filters.
  • Basic Sales Records: Establish a database tracking sales totals, calculating sum aggregates by dates and regions.
  • Simple Book Copy Inventory: Write SELECT, INSERT, and UPDATE queries to manage a database of library book copies.
🟡 Medium Projects
  • E-Commerce Database Schema: Design a relational system joining users, products, shopping carts, and checkout orders tables.
  • HR Management Database: Write queries calculating median employee salaries, group records by departments and hierarchies.
  • E-Commerce Cohort Analysis Query: Construct nested queries and CASE WHEN statements to group database orders by user registration months.
🔴 Advanced Projects
  • Hospital Ledger Analysis Project: Build an analytical script joining doctor shifts, patient check-ins, and medical records to generate detailed reports.
  • Stock Transactions Analyzer: Write queries utilizing SQL window functions to trace rolling stock prices and moving averages.
  • Rolling Sales Retention Analyzer: Write SQL window functions calculating rolling monthly sales averages and customer retention metrics.
